# Chrome Web Store Submission Copy

## Descricao Curta

Agrupa abas inativas automaticamente por site ou assunto e recolhe grupos inativos para manter o Chrome organizado.

## Descricao Completa

Idle Tab Grouper ajuda voce a manter o Chrome organizado agrupando abas que ficaram inativas por um periodo configuravel.

Principais recursos:

- Agrupa abas inativas automaticamente por alias de site ou por regras personalizadas com palavras-chave
- Permite escolher entre agrupamento automatico e modo apenas sugestao
- Recolhe grupos inativos para reduzir a bagunca visual
- Permite criar aliases de sites e regras de assunto direto no popup
- Mostra os grupos atuais para renomear, recolher ou desfazer agrupamentos
- Salva preferencias, aliases e regras usando o Chrome Sync

Casos de uso comuns:

- Organizar sessoes de pesquisa com muitas abas relacionadas
- Agrupar abas por projeto, cliente ou tema
- Manter janelas de trabalho mais limpas sem organizar tudo manualmente

Idle Tab Grouper funciona localmente no navegador e foi criado para organizar suas abas sem enviar dados de navegacao para servidores externos.

## Privacy Practices

### Single purpose

This extension's single purpose is to organize open Chrome tabs by detecting inactivity and grouping related tabs by site alias or custom subject rules.

### Data handling summary

- The extension does not send data to external servers.
- The extension does not collect or sell personal information.
- Settings are stored in Chrome storage so the extension can remember preferences and custom rules.
- Tab information is processed locally in the browser to determine inactivity and grouping behavior.

### Suggested answers for the privacy section

- Does the extension collect data? `No`
- Does the extension sell data? `No`
- Does the extension use data for purposes unrelated to the extension's single purpose? `No`
- Does the extension use data for creditworthiness or lending decisions? `No`

If the dashboard asks for a free-text explanation, you can use:

`Idle Tab Grouper processes tab metadata locally in the browser to detect inactivity and organize tabs into groups. User preferences, aliases, and custom rules are stored in Chrome storage. No browsing data is transmitted to external servers, sold, or shared for advertising or analytics.`

## Review Notes / Tester Instructions

Use this text in the reviewer instructions field:

`Open multiple regular web tabs in Chrome. Click the extension icon to open the popup. Keep the extension enabled, choose Automatic or Suggest mode, and set a short inactivity interval such as 1 minute. After the tabs remain inactive long enough, click "Agrupar agora" to trigger grouping immediately. The extension will group eligible inactive tabs by saved site aliases or keyword rules. You can also test "Recolher grupos inativos" to collapse groups when all tabs in the group are inactive. No login, paid account, or external service is required.`
